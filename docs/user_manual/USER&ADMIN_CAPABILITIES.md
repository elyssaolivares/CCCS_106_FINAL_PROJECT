# <center>Admin vs Regular User Capabilities

| Feature/Capability      | Regular User (Student/Faculty)   | Administrator                          |
| ----------------------- | -------------------------------- | -------------------------------------- |
| **Authentication**      | Log in via Google                | Log in via Google                      |
| **Dashboard View**      | Personal Dashboard (My Reports)  | System Overview Dashboard              |
| **Submit Reports**      | Create new maintenance requests  | Cannot submit, only manage             |
| **View History**        | View own submission history      | View ALL user submissions              |
| **Status Management**   | Read-only status (Pending/Fixed) | Update status (Mark as Fixed/Rejected) |
| **Category Management** | No Access                        | Add/Edit Issue Categories              |
| **System Security**     | No Access                        | View Audit Logs & User Activity        |
| **Analytics**           | No Access                        | View Report Statistics & Charts        |
-------------------------------------------------------------------------------------------------------
